Open [http://localhost:5173](http://localhost:5173) → Login: `officer` / `trinetra2026`

---

## 🛠️ Tech Stack
- **React 18 & Vite 5**: Modern SPA framework
- **Framer Motion**: Tactical HUD animations
- **Recharts**: Real-time KPI visualization
- **React Leaflet**: GIS satellite mapping (ESRI)
- **Web Speech API**: AI Voice alerts (keep-alive robust)
- **Web Audio API**: Real-time siren/klaxon generation
